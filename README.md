# 🏨 Hotel Management System
### Java Desktop Application | AIUB

A fully functional **Hotel Management System** built with **Java**, following **Object-Oriented Programming (OOP)** principles. The system provides a complete hotel administration interface for managing rooms, guests, bookings, and staff through a structured GUI-based desktop application.

---

## 📁 Project Structure

```
HOTEL-MANAGEMENT-SYSTEM/
│
├── Entity/           # Core data model classes (Room, Guest, Booking, Staff)
├── EntityList/       # Collection/list managers for each entity
├── GUI/              # Graphical User Interface panels and windows
├── Interface/        # Java interfaces defining system contracts
├── Resources/        # Images, icons, and static assets
├── Start.java        # Main entry point of the application
├── Start.class       # Compiled bytecode
└── java              # Additional Java dependencies
```

---

## ✨ Features

- 🛏️ **Room Management** — Add, update, and track room availability and types
- 👤 **Guest Management** — Register and manage guest profiles and records
- 📅 **Booking System** — Handle check-in, check-out, and reservations
- 👨‍💼 **Staff Management** — Manage hotel staff and their roles
- 🖥️ **GUI Interface** — User-friendly graphical interface built with Java Swing
- 🔗 **Interface-driven Design** — Clean abstractions via Java interfaces
- 📦 **OOP Architecture** — Organized using Entities, Lists, and GUI separation

---

## 🏗️ Architecture

| Layer | Folder | Responsibility |
|-------|--------|----------------|
| **Data Model** | `Entity/` | Defines core objects — Room, Guest, Booking, Staff |
| **Data Management** | `EntityList/` | Manages collections and CRUD operations for each entity |
| **Presentation** | `GUI/` | All UI screens, forms, and panels |
| **Contract** | `Interface/` | Java interfaces ensuring consistent implementations |
| **Assets** | `Resources/` | Images, icons, and application resources |
| **Entry Point** | `Start.java` | Launches the application |

## 🚀 How to Run

### Requirements
- Java JDK 8 or higher
- Any Java IDE (IntelliJ IDEA, Eclipse, NetBeans) or terminal

### Run from IDE
1. Clone the repository
```bash
git clone https://github.com/dnsakibss/HOTEL-MANAGEMENT-SYSTEM.git
```
2. Open the project in your Java IDE
3. Set `Start.java` as the main class
4. Click **Run**

### Run from Terminal
```bash
cd HOTEL-MANAGEMENT-SYSTEM
javac Start.java
java Start
```
